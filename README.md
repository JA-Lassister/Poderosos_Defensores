## Descrição do Projeto

"Os Poderosos Defensores da Terra" é um jogo ambientado em uma cidade futurista atacada por criaturas alienígenas.  
O jogador controla três protagonistas ao longo de cinco fases, cada uma com um estilo de gameplay diferente.

### Fases do Jogo
1. **Fase 1 — Ação em terceira pessoa**  
   Combate usando espada e arma de disparo.  
2. **Fase 2 — Corrida / perseguição**  
   Condução de veículo terrestre.  
3. **Fase 3 — Batalha aérea**  
   Controle de avião de combate.  
4. **Fase 4 — Combate marítimo**  
   Navegação e batalha em um barco.  
5. **Fase 5 — Luta com robô gigante**  
   Enfrentamento direto contra alienígenas usando um mecha.

## Funcionalidades

- Menu inicial com acesso à campanha.  
- Modo campanha passando pelas cinco fases na ordem correta.  
- Menu de seleção de fases.  
- Menu de opções com:
  - Volume ambiente  
  - Volume da música  
  - Efeito de CRT  
- Tela de instruções antes de cada fase.  
- Cinco fases completas com mecânicas distintas.

- Unity Hub 3.15.2  

## Como Abrir o Projeto

1. Abra o Unity Hub.  
2. Clique em *Add → Add project from disk*.  
3. Selecione a pasta *Poderosos-Defensores-da-Terra*.  
4. Aguarde a primeira inicialização.

## Instruções de Compilação

1. No Unity Editor, abra **File → Build Profiles**.  
2. Selecione a plataforma desejada.  
3. Se a plataforma não estiver instalada:
   - Clique em *Install with Unity Hub*.  
   - Confirme em *Install*.  
   - Aguarde e reabra o projeto.  
4. Clique em *Switch profile* se necessário.  
5. Clique em *Build* e escolha o diretório.  

## Como Executar

- No Windows: execute o arquivo `.exe`.  
- No Ubuntu (x86_64): execute o arquivo `.x86_64`.
