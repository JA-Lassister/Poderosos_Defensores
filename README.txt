DESCRIÇÃO DO PROJETO:

O jogo se passa em uma cidade futurista atacada por criaturas alienígenas vindas do espaço. O jogador controla três protagonistas em diferentes fases, cada uma possuindo um estilo de gameplay próprio:

Fase 1: Ação em terceira pessoa com espada e arma de disparo
Fase 2: Corrida/perseguição com veículo terrestre
Fase 3: Batalha aérea com avião
Fase 4: Combate marítimo em barco
Fase 5: Luta de robô gigante contra alienígena

FUNCIONALIDADES:

O jogo apresenta:
- Um menu inicial, a partir do qual é possível iniciar uma campanha, na qual o jogo avança por cada uma das fases até o final;
- Um menu de seleção de fases, para que o jogador possa escolher a fase a ser jogada;
- Um menu de opções, usado para controlar o volume ambiente, o volume da música e o efeito de CRT;
- Um menu antes de cada fase instruindo o jogador sobre os controles a serem utilizados;
- Cinco fases com estilos de gameplay distintos.

REQUISITOS DE HARDWARE:

O jogo foi testado em notebooks com as seguintes especificações (executou em todos os casos):

- CPU: Intel Core i3/i5/i7
- RAM: 8–16 GB
- GPU: Intel HD/UHD Graphics
- Sistema operacional: Windows 11 e Ubuntu 24.04.3 LTS
- 6 GB de espaço livre para o projeto (pré-compilação)

FERRAMENTAS NECESSÁRIAS:

O jogo foi desenvolvido utilizando o Unity Editor, versão 6000.2.6f2, instalado a partir do Unity Hub 3.15.2, e todas as execuções do projeto durante o desenvolvimento foram feitas usando essas ferramentas.

COMO ABRIR O PROJETO:
Abra o Unity Hub. Na aba Projects, clique em Add -> Add project from disk e carregue a pasta Poderosos-Defensores-da-Terra, presente no mesmo diretório que este README.txt. Isso criará uma nova entrada entre os projetos. Clique nesta nova entrada e aguarde até que o Unity Hub abra o projeto (demorará da primeira vez)

INSTRUÇÕES DE COMPILAÇÃO:

Uma vez aberto o Unity Editor:

- Vá em File -> Build Profiles

- Na aba "Platforms" do lado esquerdo, clique na plataforma para a qual deseja compilar o projeto.

- Caso sua plataforma não esteja carregada:
	- Aperte no botão "Install with Unity Hub"
	- No Unity Hub, aperte no botão "Install" e aguarde o término da instalação.
	- Ao término da instalação, feche o Editor e reabra o projeto.
	- No editor, vá novamente a File -> Build Profiles -> <SUA PLATAFORMA>
	
- Aperte no botão "Build". (Se estiver desabilitado, aperte em "Switch profile" primeiro)

- Escolha o diretório onde quer que o jogo seja compilado e o nome do executável e inicie o build.

- Aguarde o término da compilação.

COMO EXECUTAR:

- Vá até a pasta selecionada pra portar o executável e execute o arquivo .exe (Windows) ou .x86_64 (Ubuntu para a arquitetura x86_64)
